mcal_event_set_recur_monthly_wday -- Sets the recurrence of the streams global
event structure

  void mcal_event_set_recur_monthly_wday(int stream, int year, int month, int day, int interval)~

|mcal_event_set_recur_monthly_wday| sets the streams global event structure's
recurrence to the given value to be reoccurring on a monthly by week basis,
ending at the given date.
