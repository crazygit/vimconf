mcal_event_set_recur_weekly -- Sets the recurrence of the streams global event
structure

  void mcal_event_set_recur_weekly(int stream, int year, int month, int day, int interval, int weekdays)~

|mcal_event_set_recur_weekly| sets the streams global event structure's
recurrence to the given value to be reoccurring on a weekly basis, ending at
the given date.
