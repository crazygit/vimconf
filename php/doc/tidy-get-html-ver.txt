tidy_get_html_ver -- Get the Detected HTML version for the specified document

Procedural style:

  int tidy_get_html_ver(tidy object)~

Object oriented style:

  int tidy->getHtmlVer(tidy object)~

|tidy_get_html_ver| returns the detected HTML version for the specified tidy
{object}.

This function is not yet implemented in the Tidylib itself, so it always
return 0.
