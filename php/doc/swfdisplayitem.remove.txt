SWFDisplayItem->remove -- Removes the object from the movie

  void swfdisplayitem->remove()~

This function is EXPERIMENTAL. The behaviour of this function, the name of
this function, and anything else documented about this function may change
without notice in a future release of PHP. Use this function at your own
risk.

|swfdisplayitem->remove| removes this object from the movie's display list.

The object may be a |swfshape|, a |swfbutton|, a |swftext| or a |swfsprite|
object. It must have been added using the |swfmovie->add|.

See also |swfmovie->add|.
