SWFShape->drawLine -- Draws a line (relative)

  void swfshape->drawline(int dx, int dy)~

This function is EXPERIMENTAL. The behaviour of this function, the name of
this function, and anything else documented about this function may change
without notice in a future release of PHP. Use this function at your own
risk.

|swfshape->drawline| draws a line (using the current line style set by
|swfshape->setline|) from the current pen position to displacement
({dx},{dy}).

See also |swfshape->movepento|, |swfshape->drawcurveto|, |swfshape->movepen|
and |swfshape->drawlineto|.
